Introduction

 

Current version of AWSM - Responsive CV Template has been developed with HTML/CSS/jQuery/PHP technology and it should be compatible with all server configurations.
 

The support for this template is provided. If you are experiencing any kind of problem with installing or using this template, please feel free to contact us via contact form on our themeforest profile page http://themeforest.net/user/meks or through support@mekshq.com email.

 
AWSM - Responsive CV Template has been tested in all modern browsers: IE 8, IE 9, Opera, Firefox, Safari and Chrome and on several mobile devices and tablets
 
AWSM is a clean and simple CV/Porfolio responsive template that will help you present yourself in an appealing and professional way with responsive approach. Several different color schemes will help your profile stand out. CAPTCHA and Google Maps addon will make your site look more professional.

On which grid system is based AWSM template?

AWSM template is based on a 1140px grid system 

Explanation of the classes and markup

Class .container

Is a full width and main div in this template that allows layout to have full width of the browser. It also contains 20px padding on either side to keep content away from the edges when it becomes fluid. You can just use one big container, or use multiple to break the page horizontally. Template will look nice on big monitors and small screens.

.row

Is a row of columns. It centres them and defines the 1140px max-width. - refer to grid.css for more informations

.onecol, .twocol, .threecol, .fourcol, .fivecol, .sixcol, .sevencol, .eightcol, .ninecol, .tencol, .elevencol, .twelvecol

These are the classes for each column. They can be used in any combination within a row that adds up to twelve or less. Together with text or images.

.last

The last column within a row also needs this class. It removes the right margin so all the columns fit within the row.

  function () {

    $('#hovercard h2').append(title);

      // positioning

      var el_h = self.outerHeight();
      $('#hovercard').css({
        top: offset.top + el_h - below,
        left: offset.left - card_w/2 + el_w/2,
      }).show();  
}, 
function () {
    $('#hovercard').hide();
    $('#hovercard h2, #hovercard p.subtitle, #hovercard p.description').empty();

    How can i change awesomness level?

    This part of template is done with knob.js jQuery plugin.

    Following code:

            <input class="knob" data-fgColor="#535353" data-bgColor="#EEE" data-angleOffset="-270" data-thickness=".3" data-height="200" data-width="400" data-readOnly=true value="95">

    Here you can change following elements:

    • data-FgColor - figure data foreground color
    • data-bgColor - figure data background color
    • data-angleOffset - angle of figure data
    • data-thickness - thickness of data element
    • data-height - height of data element
    • data-width - width of data element
    • data-readOnly - is data element editable or not (leave true)
    • value - value of data element 

    If you want to change the color of template you must change data-fgColor to colors listed below :

    • black - #535353
    • blue.css - #59879F
    • red.css -#A94D59
    • purple.css - #B044F5
    • green.css - #53cf64
    • yellow.css - #EFB31B
    • turquoise.css - #53d1b2
    • orange.css - #E47445

    How can I turn off CAPTCHA?

    Template is using PHP GD library to generate form captcha. 
    If GD library is not found in your server, captcha will be automatically excluded from template.
     
    If you don't want to use form captcha please edit this line located in "ajax/generate_captcha.php":

$use_captcha =  true; //Change this to 'false' if you don't want to use captcha
